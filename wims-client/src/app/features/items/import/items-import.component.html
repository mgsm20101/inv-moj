الأعمدة المطلوبة

ورقة العمل يجب أن تُسمّى «الأصناف». الأعمدة المعلّمة (*) إلزامية.

  • * كود الصنف (اتركه فارغاً للتوليد الآلي)
  • * الاسم بالعربي
  • الاسم بالإنجليزي
  • * كود التصنيف (عقدة نهائية نشطة)
  • * نوع الصنف (مستهلك/مستديم/خطر/قابل للتلف)
  • * وحدة القياس
  • يتتبع دفعة؟ · يتتبع صلاحية؟ · يتتبع سيريال؟
  • الحد الأدنى · نقطة إعادة الطلب · الحد الأقصى
  • تصنيف الخطورة (إلزامي للخطر)
  • العمر الافتراضي (يوم) (إلزامي للقابل للتلف)
  • باركود

قاعدة الأرصدة: الحد الأدنى ≤ نقطة الطلب ≤ الحد الأقصى.
